# HashBrownandFries Portfolio

A modern, responsive landing page built with HTML, CSS, and JavaScript.

## Features

- ✨ Clean and modern design
- 📱 Fully responsive (mobile-friendly)
- 🎨 Smooth animations and transitions
- ⚡ Fast loading (no dependencies)
- 🔗 Easy to customize

## Quick Start

1. Clone or download this repository
2. Open `index.html` in your browser
3. Customize the content to match your style

## Customization

Edit the following files to personalize your site:

- **index.html** - Update your name, bio, and links
- **style.css** - Change colors, fonts, and layout
- **script.js** - Add custom functionality

### Colors

The main colors are defined in `style.css`:

```css
:root {
    --primary-color: #6c5ce7;      /* Purple */
    --secondary-color: #00b894;    /* Green */
    --text-dark: #2d3436;
    --text-light: #636e72;
    --bg-light: #f5f6fa;
    --white: #ffffff;
}
```

Change these hex values to match your brand.

## Deployment to GitHub Pages

Your site is automatically live at: `https://hashbrownandfries.github.io`

### To update your site:

1. Make changes to your files
2. Commit your changes:
   ```
   git add .
   git commit -m "Update website"
   git push
   ```
3. Changes will appear on your site within a few minutes
